# Basic Prompt for ADEs

Use this prompt in Codex, Cursor, Kiro, Windsurf, Claude Code, or similar ADEs.

```text
This project follows CDAD — Context-Driven AI Development.

Before making implementation decisions, read:

- AGENTS.md
- .cdad/project-context.md
- .cdad/ai-rules.md
- context/vision.md
- context/architecture.md
- context/principles.md
- context/constraints.md

Do not modify L0 files directly.

If an architecture change is needed, return a Proposed Architecture Change and wait for approval.
```
